## Installation in Termux
`yes|pkg update && pkg upgrade`
`termux-setup-storage`
`yes|pkg install git`
`yes|pkg install fish`
`git clone https://github.com/Darkmux/cyberspy`
`cd cyberspy`
`chmod 777 *.sh`
`bash cyberspy.sh`
